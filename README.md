# Student Satisfaction Survey Analysis (2025)

This is a Shiny application for analyzing the Student Satisfaction Survey for the Agricultural Sciences Program at UWA.

## Files
- `app.R`: The main Shiny application script.
- `Student Satisfaction Survey for Agricultural Sciences Program - Form responses.csv`: The raw survey data.

## Deployment
This repository is structured to be deployed to Posit Cloud or shinyapps.io.
